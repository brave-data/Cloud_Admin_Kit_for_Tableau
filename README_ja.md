# Cloud Admin Kit for Tableau

[![License: Polyform Noncommercial](https://img.shields.io/badge/License-Polyform_NC-blue.svg)](LICENSE)
[![Python 3.11+](https://img.shields.io/badge/Python-3.11%2B-blue.svg)](https://www.python.org/)

> **English README**: [README.md](README.md)

Tableau Cloudサイトの**一元管理・変更追跡・メンテナンス効率化**を実現するローカルWebアプリです。
Python + FastAPIで構築され、すべて手元のマシン上で動作します。

---

## なぜこのツールが必要か

Tableau Cloud の管理画面は機能ごとに分散しており、「どのデータソースがどのダッシュボードで使われているか」「昨日と今日で何が変わったか」「誰が長期間ログインしていないか」を把握するには、複数画面を行き来する必要があります。

**Cloud Admin Kit for Tableau** は、管理者が日常のメンテナンスで本当に必要な情報を1画面に集約し、Tableau Cloudの管理を楽にするためのツールです。

---

## 主な機能

### 1. サイト全体をダッシュボードで把握

![ダッシュボード概要](docs/screenshots/01_dashboard.png)

ページを開くだけで、サイト全体の状況が2段のサマリーカードで一目でわかります。

**上段 — サイト合計:**
- **ワークブック / Prepフロー / データソース / ビュー / ユーザー**

**下段 — アラートカード（閾値を変えると即時反映）:**
- **N日以上未閲覧** — 整理・削除を検討すべきコンテンツ数
- **N日以上未ログイン** — ライセンス最適化の候補ユーザー数
- **N日以上未更新データソース** — 最終更新が閾値を超えたデータソース数
- **未参照データソース** — どのワークブックからも参照されていない公開済みデータソース数
- **⚙ 閾値を設定** — アラートの日数を個人ごとにカスタマイズ、ブラウザに保存

カードをクリックするだけで対応する一覧タブへ即ジャンプ。

---

### 2. データソースと紐づきをひと目で確認

![データソース一覧](docs/screenshots/02_datasources.png)

データソース一覧では、各データソースの**プロジェクト・オーナー・タイプ・認定状態・最終更新からの経過日数**を一覧表示。

- 経過日数が長いデータソース（赤表示）を即座に特定
- **未参照**バッジ付きのデータソースは、現在どのワークブックからも接続されていない整理候補
- **参照数**列でワークブックからの接続数を確認
- 各行を展開すると、同プロジェクト内のワークブック一覧を表示

---

### 3. 人気コンテンツのランキング把握

![Top ビュー](docs/screenshots/03_topviews.png)

累計ビュー数 Top 10 を棒グラフで可視化。

- どのダッシュボードが最も使われているかを即座に把握
- 利用実績に基づくコンテンツ整理・優先メンテナンスの判断材料に

---

### 4. 日次の変更を自動追跡（差分比較）

![差分比較](docs/screenshots/05_diff.png)

「昨日と比べて今日何が変わったか」を自動で検出します。

- 🟢 **新規追加**: 新しく作成されたWB・DS・フロー・ユーザー
- 🟡 **更新**: 名前・プロジェクト・オーナーの変更
- 🔴 **削除**: 前日存在していたが今日なくなったコンテンツ
- **KTWタグ**を付けたワークブックは計算フィールドの追加・削除・数式変更まで追跡

変更通知機能のないTableau Cloudで、**誰かが誤ってコンテンツを削除したり、意図しない変更が加えられた場合でも、翌日の確認で即座に気づける**ようになります。

---

### 5. 更新スケジュールの監視

![スケジュール管理](docs/screenshots/06_schedules.png)

データの鮮度を保つために欠かせないスケジュール管理。

- 全抽出更新スケジュールを一覧表示
- 次回実行時刻が過去になっているものを**「期限超過」バッジ**でハイライト → 更新が止まっているデータソースを即特定

---

### 6. ワークブックのリビジョン差分比較

ワークブックの任意の2リビジョンを比較し、何が変わったかをピンポイントで確認できます。

- ワークブックを選択し、**Base**（比較元）と **Head**（比較先）のリビジョン番号を指定
- **追加 / 削除 / 変更** を以下のカテゴリ別に表示：
  - **計算フィールド** — フィールド名・データソース・変更前後のフォーミュラ
  - **フィルター** — カテゴリ・数値・相対日付・上位N件フィルターの変更
  - **接続データソース** — データソースの追加・削除
  - **シート** — ワークシート・ダッシュボード・ストーリーの追加・削除
- 取得結果はリビジョンペアごとにキャッシュ — 2回目以降は即座に表示
- 最新リビジョンとひとつ前を比較して、意図しない計算フィールドの変更をその日のうちに検出

---

### 7. 計算フィールドの依存関係を分析

![計算フィールド分析](docs/screenshots/04_fields.png)

ワークブックの計算フィールドをダウンロード・解析し、フィールド間の依存関係をSankeyチャートで可視化。

- どの計算フィールドがどのフィールドに依存しているかを視覚的に把握
- 複雑な計算ロジックのメンテナンス・引き継ぎに役立つ

---

### 8. バックグラウンド自動更新

アプリがバックグラウンドで定期的（デフォルト：**30分ごと**）にTableau Cloudからデータを自動再取得します。手動で更新ボタンを押さなくても、常に最新に近いデータが表示されます。

- `.env` の `REFRESH_INTERVAL_MINUTES` で間隔を変更可能（APIレート制限を考慮して15分以上を推奨）
- ステータスバーに最終取得時刻を表示 — データの鮮度が一目でわかる
- 取得中は重複実行をスキップ — 安全に動作

---

### 9. アラート閾値のカスタマイズ

ダッシュボード右下の **⚙ 閾値を設定** カードをクリックすると、各アラートの日数を個別に変更できます。

| アラートカード | デフォルト | カウント対象 |
|---|---|---|
| 未閲覧 | 180日 | 閾値を超えて閲覧実績のないワークブック |
| 未ログイン | 90日 | 閾値を超えてログインがないユーザー |
| 未更新DS | 30日 | 最終更新日が閾値を超えたデータソース |

設定はブラウザの `localStorage` に保存され、変更後は即時反映。サーバー再起動不要です。

---

### 10. ダークモード対応

![ダークモード](docs/screenshots/07_dark_mode.png)

ライト / ダークモードを切り替え可能。長時間の管理作業でも目に優しい表示で利用できます。日本語 / 英語の言語切替にも対応しています。

## クイックスタート

```bash
git clone https://github.com/brave-data/Cloud_Admin_Kit_for_Tableau.git
cd Cloud_Admin_Kit_for_Tableau
python3 -m venv .venv && source .venv/bin/activate   # Python 3.11以上が必要
pip install -r requirements.txt
cp .env.example .env   # Tableau Cloudの認証情報を入力
python main.py
```

ブラウザで **http://localhost:8000** を開き、**「↻ 更新」ボタン** を押すとデータ取得が始まります。

**→ 詳細なセットアップ手順・PATの作成方法・トラブルシューティング: [SETUP_ja.md](SETUP_ja.md)**

## KTWタグ — 計算フィールド変更監視

Tableau Cloud のワークブックに `KTW` タグを付けると、計算フィールドの自動監視が有効になります。

- 更新のたびに最大10件のKTWタグ付きWBをダウンロード・解析
- フィールドの**追加・削除・数式変更**が差分比較タブの「計算フィールド [KTW監視]」カテゴリに表示
- 「誰かが計算フィールドをこっそり変えた」を翌日に自動検出

## APIエンドポイント

アプリ起動中は `http://localhost:8000/docs` でSwagger UIが利用できます。

| エンドポイント | 説明 |
|--------------|------|
| `GET /api/status` | 取得ステータス・最終更新日時・部分取得エラーの警告 |
| `GET /api/summary` | サイト全体のサマリー統計 |
| `GET /api/workbooks` | ワークブック一覧 |
| `GET /api/datasources` | データソース一覧（`is_ghost` フラグ含む） |
| `GET /api/views` | ビュー一覧と累計ビュー数 |
| `GET /api/users` | ユーザー一覧 |
| `GET /api/flows` | Prepフロー一覧 |
| `GET /api/schedules` | 更新スケジュール一覧 |
| `GET /api/flows/{id}/connections` | Prepフローの入出力接続情報 |
| `GET /api/workbooks/{id}/fields` | ワークブックの計算フィールド分析 |
| `GET /api/workbooks/{id}/revisions` | ワークブックのリビジョン一覧 |
| `GET /api/workbooks/{id}/revision-diff` | 2リビジョン間の差分（`?base=N&head=M`；省略時は最新 vs 1つ前） |
| `GET /api/ktw-fields` | KTWタグ付きWBの計算フィールド（最大10件） |
| `POST /api/refresh` | Tableau Cloudからデータを再取得 |
