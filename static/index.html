使い方

ページ上部のサマリーカードにTableau Cloudサイト全体の統計が表示されます。

  • 各カードをクリックすると、対応する一覧タブへ移動します
  • 右上の「更新」ボタンを押すとTableau CloudのAPIからデータを再取得します（初回は必ず押してください）
  • 180日以上未閲覧 / 90日以上未ログインのカードは要注意コンテンツ数を示します

Summary cards at the top show site-wide statistics from your Tableau Cloud site.

  • Click any card to navigate to the corresponding list tab
  • Click ↻ Refresh to fetch fresh data from the Tableau Cloud API (required on first use)
  • The Not Viewed 180+ Days and Inactive 90+ Days cards highlight content at risk

  • 各タブはサマリーカードのクリック、またはナビタブから開けます
  • テーブルの列ヘッダーをクリックでソート、検索ボックスでフィルタできます
  • ワークブック行の「フィールド分析」ボタンを押すと計算フィールド分析タブへジャンプします
  • 「認定済み」バッジはデータソースの品質認定状態を示します
  • 「タグ」列に KTW タグがついたワークブックは差分比較の監視対象になります
  • Open tabs by clicking summary cards or the navigation tabs
  • Click column headers to sort; use the search box to filter
  • Click the Analyze Fields button on a workbook row to jump to the Calculated Fields tab
  • The Certified badge reflects the data quality certification status of a data source
  • Workbooks tagged KTW are monitored for calculated field changes in the Changes tab

Tableau Cloudに設定されているデータ更新スケジュール一覧です。

  • 「次回実行」列の期限超過バッジは更新が止まっている可能性を示します
  • 更新種別（完全更新 / 差分更新）がアイコンで区別されます

Lists all data refresh schedules configured in Tableau Cloud.

  • An Overdue badge in the Next Run column indicates a potentially stalled refresh
  • Refresh type (Full Refresh / Incremental) is shown as a badge

ワークブックをダウンロードして計算フィールドの依存関係を可視化します。

  • ワークブック一覧の「フィールド分析」ボタン、またはこのタブのドロップダウンからWBを選択します
  • Sankeyチャートでフィールド間の依存関係を視覚的に確認できます
  • 分析結果はキャッシュされるため、2回目以降は即時表示されます
  • KTWタグ付きWBは更新ボタン押下時に自動スキャンされ、計算フィールドの追加・削除・数式変更が差分タブに表示されます

Downloads a workbook and visualizes calculated field dependencies.

  • Select a workbook using the Analyze Fields button in the workbook list or the dropdown in this tab
  • The Sankey chart shows field dependency relationships visually
  • Results are cached — subsequent runs are instant
  • Workbooks tagged KTW are auto-scanned on Refresh; field additions, deletions, and formula changes appear in the Changes tab

「前日」と「今回の更新」の間に何が変わったかを表示します。

  • 🟢 新規追加：前回になかったWB・DS・フロー・ユーザー
  • 🟡 更新：名前・プロジェクト・シート構成・KTW計算フィールドの変化
  • 🔴 削除：前回あったが今回なくなったもの
  • 初日は比較元がないため表示されません。翌日以降に差分が表示されます
  • KTWタグ付きWBは計算フィールドの変化も「計算フィールド [KTW監視]」カテゴリに表示されます

Shows what changed between the previous day's snapshot and the current data.

  • 🟢 New: workbooks, data sources, flows, or users that didn't exist before
  • 🟡 Updated: name, project, sheet structure, or KTW field changes
  • 🔴 Deleted: items that existed previously but are now gone
  • No diff is shown on the first day — comparison requires a previous day's baseline
  • Workbooks tagged KTW also show calculated field changes under the "Calc Fields [KTW Watch]" category
