基本操作
Refresh ボタン 右上
Tableau Cloud から最新データを取得します。通常 10〜30 秒かかります。
押すたびにスナップショットが保存され、次回以降の差分比較に使用されます。
JA / EN ボタン 右上
画面全体の表示言語を日本語 / 英語で切り替えます。設定はページを閉じても記憶されます。
ダークモードボタン 右上
画面をダークモード / ライトモードに切り替えます。目に優しい夜間作業向けです。
サマリカード 画面上部
WB数・DS数・フロー数・ビュー数・ユーザー数をひと目で確認できます。
180日以上未閲覧と90日以上未ログインのカウントはメンテナンス指標です。
ワークブックタブ
一覧テーブル
全WBのプロジェクト・オーナー・最終更新日・未閲覧日数・サイズ・タグを表示します。
列ヘッダーをクリックするとソート、検索ボックスで絞り込みができます。
▶ 行を展開（クリック）
行をクリックすると、そのWBに含まれるビュー（シート）の一覧が展開表示されます。
未閲覧日数の赤ハイライト
180日以上閲覧されていないWBは未閲覧日数のセルが赤く表示されます。アーカイブや削除の検討サインです。
データソースタブ
一覧テーブル
タイプ・プロジェクト・オーナー・最終更新・認定状態を確認できます。
認定バッジ
「認定済み」のデータソースには緑のバッジが表示されます。未認定のものは品質確認を推奨します。
ビュータブ（使用状況）
累計アクセス数
各シートの累計アクセス数（total_views）を確認できます。使われていないシートの洗い出しに活用してください。
ユーザータブ
ライセンス（サイトロール）別一覧
ライセンス種別・最終ログイン日・未ログイン日数を確認できます。
未ログイン日数の赤ハイライト
90日以上ログインのないユーザーはハイライト表示。ライセンス費用の最適化に役立てます。
Prep フロータブ
▶ 行を展開（クリック）
行をクリックするとフローの接続情報（接続先DB・サーバーなど）が展開表示されます。
スケジュールタブ メニュー内
エクストラクト更新スケジュール
各DS・WBのエクストラクト更新の頻度・次回実行予定時刻を一覧表示します。
ナビバー右端「Schedules」または上部ナビから開きます。
Top ビュータブ メニュー内
アクセス数 Top 10 グラフ
全ビューの中でアクセス数が多い上位10件を横棒グラフで表示します。よく使われているコンテンツが一目でわかります。
計算フィールドタブ メニュー内
ワークブック選択 → フィールド解析
WBを選択すると .twb/.twbx をダウンロードして計算フィールドを抽出します。
  • 計算フィールド一覧（フィールド名・数式・データソース）
  • フィールド間の依存関係をサンキーチャートで可視化
差分比較タブ メニュー内
比較タイミング
前回 Refresh 時のデータ vs 今回 Refresh 時のデータを比較します。
初回は「まだスナップショットがありません」と表示され、2回目の Refresh 後から差分が確認できます。
🟢 新規追加セクション
前回以降に新しく追加されたWB・DS・フロー・ユーザーを表示します。
🟡 更新セクション
変更があったアイテムを、何がどう変わったか（旧値 → 新値）で表示します。
  • ワークブック（設定変更）: 名前・プロジェクト・オーナーの変更
  • ワークブック（シート変更）: シートの追加・削除
  • 計算フィールド [KTW監視]: KTWタグWBの計算フィールド追加・削除・数式変更
🔴 削除セクション
前回以降に削除されたWB・DS・フロー・ユーザーを表示します。
KTW監視タグ
KTWタグの付け方
Tableau Cloud でワークブックに KTW タグをつけると、そのWBが計算フィールド監視対象になります。
  • Tableau Cloud の WB 詳細ページ → 「タグを追加」→ KTW と入力
  • 最大10件まで監視対象に登録できます
自動スキャンのタイミング
Refresh ボタンを押すたびに、バックグラウンドで KTWタグWBをダウンロードして計算フィールドを比較します。差分は差分比較タブに反映されます。
